Установка Heroku

Заметки
  • Если появились иные ошибки — @heroku_talks
  • Если UserLand зависает на «Проверка файлов...» — перезагрузи телефон
  • Повторный запуск используй после первой установки
  • Иногда терминал покажет пустую строку userland@localhost# — просто продублируй команду ещё раз
Arch Linux UserLand
Установка
bash 
su root
pacman -Syu python python-pip git gcc glibc --noconfirm
python3 -m venv venv && source venv/bin/activate
git clone https://github.com/coddrago/Heroku && cd Heroku && pip install aiohttp==3.9.5 && pip install aiogram && pip install -r requirements.txt && python3 -m heroku --no-web
Повторный запуск
bash 
su root && cd Heroku && source venv/bin/activate && python3 -m heroku --no-web
Ubuntu UserLand
Установка
bash 
su root
apt install python3.14-venv -y
python3 -m venv venv && source venv/bin/activate
apt update && apt install git -y && apt install python3-pip -y && git clone https://github.com/coddrago/Heroku && cd Heroku && pip install -r requirements.txt && python3 -m heroku --no-web
Повторный запуск
bash 
su root && cd Heroku && source venv/bin/activate && python3 -m heroku --no-web
